# Parallel-and-High-computing
This repo is designed to submit assignment.

## Homework structure
1) For each homework there is seperate folder eg `Homework1`
2) In each homework there can be multiple assignments folder if homework has more than one assignment
3) In each assigment folder there is homework file with `.c` extension, output folder/file which contain screenshot of my assignment result and explaination.md file which contain solution description


## Example of folder structure
